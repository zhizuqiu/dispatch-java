Usage of dispatch-up:
  -a string
    	dispatch server地址，例如：
  -config string
    	dispatch配置文件路径
  -d string
    	要上传到的目录路径，例如：/temp/
  -f string
    	要上传的文件路径，例如：./nginx.conf
  -h	show help.

例如：
dispatch-up -f ./some.zip
dispatch-up -a  -f ./some.zip
dispatch-up -a  -d /temp/ -f ./some.zip
